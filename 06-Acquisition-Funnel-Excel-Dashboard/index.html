Power Query Pipeline

  • CSV raw spend data ingestion and header promotion.
  • Removal of unnecessary structural columns (e.g. redundant system IDs).
  • Platform and campaign segment extraction from text strings.
  • Text cleaning using Trim, Clean, and case standardizations.
  • Enforcement of clean data types (Decimal, Integer, Date).

Formula Architecture & aggregate safety

  • `IFERROR` wrapping prevents division-by-zero errors (`#DIV/0!`) for inactive campaigns.
  • Weighted averages prevent the "average of averages" statistical error in pivot summaries.
  • Exact ROI calculations based on true cost-per-lead and customer-acquisition metrics.
  • Interactive slicers synchronize metrics in real-time.
